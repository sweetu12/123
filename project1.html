3D Printed Custom Chess Piece

This project involves designing and 3D printing a custom chess piece using SolidWorks and FDM 3D printing technology.

Project Overview

The project aimed to create a custom chess piece that was both aesthetically pleasing and functional. We used SolidWorks for the design and an FDM printer for the actual production. The goal was to experiment with design optimization for strength and material usage.
